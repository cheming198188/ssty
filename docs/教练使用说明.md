# 神兽体育训练课件与报告平台教练使用说明

适用对象：门店教练、课程主管、店长

适用场景：早教、私教课、小班课/团课、专项（闯关）训练、跑步训练

## 一、系统用途

本系统用于帮助教练完成以下工作：

- 建立会员档案
- 根据训练对象、目标和周期生成训练计划
- 查看每周周期安排和单节课教案
- 记录每次训练过程
- 生成家长图文版训练汇报
- 上传训练照片和短视频

## 二、登录系统

1. 打开系统网址
2. 输入账号和密码
3. 点击“登录进入系统”

说明：

- `管理员` 可以删除学员档案和训练报告
- `教练` 可以正常建档、排课、带课、记录训练，但不能删除高风险数据

如果忘记密码，请联系管理员重置。

## 三、学员建档

进入系统后，先完成“学员建档与课程设定”。

必填和常用字段包括：

- 学员姓名
- 训练对象：幼儿 / 青少年 / 成人
- 年龄
- 性别
- 课程类型
- 训练目标
- 训练周期
- 每周课次
- 单次课时

建议尽量补充这些信息，便于生成更准确的课程：

- 学校、年级
- 家长姓名、家长电话
- 既往训练经历
- 运动兴趣 / 偏好项目
- 当前评估情况
- 基础测试数据
- 客户需求
- 病史 / 医疗注意事项
- 既往伤病记录
- 注意事项 / 运动限制
- 性格与课堂沟通备注

填写完成后，点击“保存档案”。

## 四、查找与新建会员

左侧“学员档案列表”可用于快速查找和切换会员。

使用方法：

- 输入会员姓名或家长手机号，可快速搜索
- 点击某个会员卡片，可自动回填表单
- 点击“新建学员”，可清空当前表单并录入新会员

## 五、生成训练计划和课件

完成建档后，点击“生成定制训练课件”。

系统会自动生成：

- 训练计划摘要
- 周期安排
- 单节课教案
- 课程模板
- 家长沟通模板

教练重点查看区域：

### 1. 周期安排

按周查看训练重点、主线和课次分布。

### 2. 单节课教案

点击某一节课后，可以查看：

- 动作类别
- 执行顺序
- 时间分配
- 组数 / 次数
- 间歇要求
- 强度要求
- 动作要求
- 安全提示
- 降阶版 / 标准版 / 进阶版

### 3. 课程模板

适合教练提前备课，查看整节课结构和视频示范入口。

## 六、打印单节课教案

在“单节课教案”区域中：

1. 先点击某一节课
2. 确认右侧出现完整教案
3. 点击“打印这节课”

系统会打开独立打印页，适合 A4 打印。

适用场景：

- 上课携带纸质课件
- 店内存档
- 课程复盘

## 七、记录单次训练

每次课后，进入“单次训练记录”区域填写：

- 训练日期
- 课次时长
- 课堂投入度
- 主观强度 RPE
- 本次完成内容
- 教练备注
- 家庭练习建议
- 训练照片 / 短视频

填写完成后，点击“保存本次训练报告”。

## 八、生成家长汇报

保存训练报告后，系统会自动生成：

- 家长图文版预览
- 本阶段进展图表
- 本周与上周对比
- 周期目标完成度
- 本周主线进展
- 下节课衔接建议
- 家长配合建议
- 训练素材展示

如需发家长：

1. 在“家长图文版预览”中检查内容
2. 点击“打开微信长图版”
3. 截图或直接转发给家长

## 九、最近训练汇报

页面下方“最近训练汇报”可用于快速回看近期已保存报告。

适合：

- 课后复盘
- 向家长二次沟通
- 查看近期训练进展

## 十、删除功能

删除功能仅管理员可用。

### 1. 删除学员

在“学员档案列表”中点击“删除”。

删除后会同时清理：

- 学员档案
- 该学员关联的训练报告
- 关联照片 / 视频素材

### 2. 删除训练报告

在“最近训练汇报”中点击“删除”。

删除后会同时清理：

- 该条训练报告
- 该报告关联的照片 / 视频素材

请谨慎操作，删除后无法恢复。

## 十一、推荐使用流程

建议教练按以下顺序使用：

1. 新建或搜索会员
2. 完成建档与课程设定
3. 生成定制训练课件
4. 查看周期安排和单节课教案
5. 课后填写单次训练记录
6. 保存训练报告
7. 打开家长长图版并发送家长

## 十二、常见问题

### 1. 为什么看不到系统内容？

请先登录账号。

### 2. 为什么不能删除学员或报告？

当前登录账号可能是“教练”角色，不具备删除权限，请联系管理员。

### 3. 为什么课程内容和会员不匹配？

请先检查：

- 年龄是否填写正确
- 训练对象是否选择正确
- 课程类型是否正确
- 训练目标是否正确
- 评估和限制条件是否填写完整

### 4. 为什么上传素材后看不到？

请确认：

- 是否真的点击了“保存本次训练报告”
- 图片或视频文件是否过大
- 网络是否正常

### 5. 为什么网址打开有点慢？

当前使用的是 Render 免费实例，长时间无人访问后会休眠，首次打开可能需要等待几十秒。

## 十三、培训建议

建议门店培训时按以下方式安排：

- 第一步：先教登录和建档
- 第二步：再教生成训练计划与查看教案
- 第三步：最后教课后记录和家长报告发送

新人教练培训时，建议先用 1 名示例会员完整演练一次流程。

## 十四、门店管理建议

管理员建议定期检查：

- 学员档案是否完整
- 训练报告是否按时填写
- 素材上传是否规范
- 是否存在重复会员
- 是否存在不再需要的旧报告和旧素材
