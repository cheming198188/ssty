# 神兽体育训练课件与报告平台教练使用说明

适用对象：门店教练、课程主管、店长  
适用场景：早教、私教课、小班课/团课、专项（闯关）训练、跑步训练  
适用训练对象：幼儿 / 青少年 / 成人

## 一、系统用途

本系统用于帮助教练完成以下工作：

- 登录教练工作台并进入个人工作区
- 建立会员档案
- 根据训练对象、训练目标和训练周期生成训练计划
- 查看每周周期安排和单节课教案
- 自动将当前选中的课次内容带入课后记录
- 记录每次训练过程
- 生成家长图文版训练汇报
- 生成微信长图版家长报告
- 导出家长报告 PDF
- 上传训练照片和短视频

## 二、登录系统与账号权限

1. 打开系统网址
2. 输入账号和密码
3. 点击“登录进入系统”

权限说明：

- `管理员`：可查看全部数据，并可删除学员档案和训练报告
- `教练`：可建档、排课、带课、记录训练，但不能删除高风险数据
- 教练登录后，系统默认仅显示自己名下的学员和训练报告
- 系统已支持登录状态保持，服务重启后通常不会立即掉线

如忘记密码，请联系管理员重置。

## 三、学员建档与课程设定

进入系统后，先完成“学员建档与课程设定”。

常用必填字段包括：

- 学员姓名
- 训练对象：幼儿 / 青少年 / 成人
- 年龄
- 性别
- 课程类型
- 训练目标
- 训练周期
- 每周课次
- 单次课时

建议尽量补充以下信息，以便系统生成更准确的训练方案：

- 学校、年级
- 家长姓名、家长电话
- 既往训练经历
- 运动兴趣 / 偏好项目
- 当前评估情况
- 基础测试数据
- 客户需求
- 病史 / 医疗注意事项
- 既往伤病记录
- 注意事项 / 运动限制
- 性格与课堂沟通备注

填写完成后，点击“保存档案”。

## 四、查找、新建与查看归属学员

左侧“学员档案列表”可用于快速查找和切换会员。

使用方法：

- 输入会员姓名或家长手机号，可快速搜索
- 点击某个会员卡片，可自动回填表单
- 点击“新建学员”，可清空当前表单并录入新会员
- 教练账号登录后，列表默认仅显示自己负责的会员
- 管理员账号登录后，可查看全部会员

## 五、生成训练计划和课件

完成建档后，点击“生成定制训练课件”。

系统会自动生成：

- 训练计划摘要
- 周期安排
- 单节课教案
- 课程模板
- 家长沟通模板

教练重点查看区域如下：

### 1. 周期安排

按周查看训练重点、训练主线和课次分布。

### 2. 单节课教案

点击某一节课后，可以查看：

- 动作类别
- 执行顺序
- 时间分配
- 组数 / 次数
- 间歇要求
- 强度要求
- 动作要求
- 安全提示
- 降阶版 / 标准版 / 进阶版

### 3. 课程模板

适合教练提前备课，查看整节课结构和动作示范入口。

## 六、打印单节课教案

在“单节课教案”区域中：

1. 先点击某一节课
2. 确认右侧出现完整教案
3. 点击“打印这节课”

系统会打开独立打印页，适合 A4 纸张打印。

适用场景：

- 上课时携带纸质教案
- 门店内部留档
- 教练课程复盘

## 七、记录单次训练

每次课后，进入“单次训练记录”区域填写：

- 训练日期
- 课次时长
- 课堂投入度
- 主观强度 RPE
- 本次完成内容
- 教练备注
- 家庭练习建议
- 训练照片 / 短视频

说明：

- 当你在“周期安排”中点击某一节课时，系统会自动将该课次内容带入“本次完成内容”
- 如果课后执行内容与原教案有调整，教练可根据实际情况再手动修改

填写完成后，点击“保存本次训练报告”。

## 八、生成家长汇报

保存训练报告后，系统会自动生成：

- 家长图文版预览
- 本阶段进展图表
- 本周与上周对比
- 周期目标完成度
- 本周主线进展
- 下节课衔接建议
- 家长配合建议
- 训练素材展示

发送给家长时，可按以下方式操作：

1. 在“家长图文版预览”中检查内容
2. 点击“重新生成长图”，打开微信长图版
3. 截图或直接转发给家长

如需导出纸质版或电子版：

1. 在“家长图文版预览”中点击“导出 PDF”
2. 系统会打开可打印页面
3. 在浏览器中保存为 PDF，或直接打印

## 九、最近训练汇报

页面下方“最近训练汇报”可用于快速回看近期已保存的报告。

目前支持：

- 按学员姓名搜索
- 按训练目标或日期搜索
- 按“是否含素材”筛选
- 按投入度筛选
- 分页查看
- 点击某条报告后，直接在右侧查看家长预览

适用场景：

- 课后复盘
- 向家长进行二次沟通
- 查看近期训练进展
- 快速找回某一名学员的课后记录

## 十、删除功能

删除功能仅管理员可用。

### 1. 删除学员

在“学员档案列表”中点击“删除”。

删除后会同时清理：

- 学员档案
- 该学员关联的训练报告
- 关联的照片 / 视频素材

### 2. 删除训练报告

在“最近训练汇报”中点击“删除”。

删除后会同时清理：

- 该条训练报告
- 该报告关联的照片 / 视频素材

请谨慎操作，删除后无法恢复。

## 十一、推荐使用流程

建议教练按以下顺序使用：

1. 登录系统
2. 搜索已有会员或新建会员
3. 完成建档与课程设定
4. 生成定制训练课件
5. 查看周期安排和单节课教案
6. 上课前打印教案，或直接按页面带课
7. 课后点击本次课次，自动带入训练记录
8. 补充教练备注、家庭练习和训练素材
9. 保存训练报告
10. 重新生成长图或导出 PDF 发给家长

## 十二、常见问题

### 1. 为什么看不到系统内容？

请先登录账号。

### 2. 为什么只能看到自己的一部分数据？

当前登录账号可能是“教练”角色。教练默认只显示自己负责的学员和训练报告，这是系统的数据归属设置。

### 3. 为什么不能删除学员或报告？

当前登录账号可能是“教练”角色，不具备删除权限，请联系管理员处理。

### 4. 为什么课程内容和会员不匹配？

请先检查以下内容：

- 年龄是否填写正确
- 训练对象是否选择正确
- 课程类型是否选择正确
- 训练目标是否选择正确
- 评估和限制条件是否填写完整

### 5. 为什么上传素材后看不到？

请确认：

- 是否确实点击了“保存本次训练报告”
- 图片或视频文件是否过大
- 当前网络是否正常

### 6. 为什么网址打开有点慢？

当前系统部署在 Render 免费实例上，长时间无人访问后会进入休眠状态，因此首次打开可能需要等待几十秒。

## 十三、培训建议

建议门店培训时按以下方式安排：

- 第一步：先培训登录和学员归属查看
- 第二步：再培训建档、生成训练计划与查看教案
- 第三步：培训课后记录自动带入和家长报告发送
- 第四步：补充最近训练汇报的搜索、筛选与分页使用

新人教练培训时，建议先以 1 名示例会员完整演练一次流程。

## 十四、门店管理建议

管理员建议定期检查：

- 学员档案是否完整
- 训练报告是否按时填写
- 素材上传是否规范
- 是否存在重复会员
- 是否存在不再需要的旧报告和旧素材
- 教练归属是否准确
